# git helper

```PowerShell
mkdir [myProject]
git clone https://github.com/zeged85/homesystem.git
cd homesystem
```

# add/edit files

```PowerShell
echo "bla bla" >> test.txt
```

# chek status
```PowerShell
git status
```


# then:
```PowerShell
git add test.txt
```

# commit:
```PowerShell
git commit -m "new file test.txt added"
```

# finally:
```PowerShell
git push origin master
```
